# Online Bookstore Sales & Inventory Analysis ( Power BI)

## Project Overview
This project analyzes an online bookstore dataset to understand sales performance, customer insights, and inventory status. An interactive Power BI dashboard was created to visualize key business metrics and generate actionable insights.

## Tools Used
- Power BI
- SQL
- Excel/CSV Dataset

## Dataset
The project contains three datasets:
- Books.csv
- Customers.csv
- Orders.csv

## Dashboard Pages
### 1. Overview
- Total Revenue
- Total Books Sold
- Available Book Stock
- Top Books Sold
- Revenue by Genre

### 2. Customer Analysis
- Customer-related insights and analysis

### 3. Inventory & Stock Analysis
- Available Stock by Genre
- Inventory Stock Status
- Low Stock Analysis
- Stock Insights

## Key Insights
- Identified top-performing books and authors
- Analyzed revenue contribution by genre
- Monitored available inventory levels
- Identified low stock books requiring attention

## Project Files
- PowerBI/ → Power BI dashboard file
- SQL/ → SQL queries
- Dataset/ → Source CSV files
- Screenshots/ → Dashboard images

## Conclusion
This project demonstrates SQL analysis, data visualization, and business insight generation using Power BI.
